# Factorial

## 问题描述

给定一个非负整数 `n`，计算 `n!` 的值。

其中：
- `0! = 1`
- 当 `n > 0` 时，`n! = 1 * 2 * 3 * ... * n`

约定：
- 输入 `n` 为非负整数。
-  0 <= `n` <= 10。
- 参考实现假设结果不会超出 `int` 的表示范围。

## 正确代码

```c
int fac(int n) {
    int i;
    int res = 1;

    for (i = 1; i <= n; ++i) {
        res = res * i;
    }

    return res;
}
```

## 说明

这段代码使用一个累乘循环计算阶乘：
- `res` 表示当前已经累乘得到的结果
- 循环变量 `i` 从 `1` 递增到 `n`
- 每一步把当前的 `i` 乘到 `res` 上

当循环结束时，`res` 就等于 `n!`。
