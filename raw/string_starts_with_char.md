# String Starts With Char

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s` 和字符 `c`，判断字符串是否以字符 `c` 开头。如果是返回 `1`，否则返回 `0`。

约定：

- 输入是合法的 C 风格字符串
- 空字符串不以任何普通字符开头
- 函数不修改字符串
- 返回值只使用 `0` 或 `1`

## 正确代码

```c
int string_starts_with_char(char *s, char c) {
    if (s[0] == '\0') {
        return 0;
    }
    if (s[0] == c) {
        return 1;
    }
    return 0;
}
```

## 说明

这道题适合验证“字符串首字符判断 + 空串分支”模式。
