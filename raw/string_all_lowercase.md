# String All Lowercase

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s`，判断其中所有字符是否都在 `'a'` 到 `'z'` 之间。

约定：

- 输入是合法的 C 风格字符串
- 函数不修改字符串
- 如果所有字符都是小写字母，返回 `1`
- 否则返回 `0`

## 正确代码

```c
int string_all_lowercase(char *s) {
    int i = 0;

    while (s[i] != '\0') {
        if (s[i] < 'a' || s[i] > 'z') {
            return 0;
        }
        i++;
    }

    return 1;
}
```

## 说明

这道题适合验证“字符串扫描 + 区间判断”模式。
