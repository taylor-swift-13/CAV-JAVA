# Sign

Implement a Java method that returns the sign of an integer.

Target method:

```java
public static int sign(int x)
```

Behavior:

- If `x > 0`, return `1`.
- If `x == 0`, return `0`.
- If `x < 0`, return `-1`.

Requirements:

- The generated Java class should be named `Sign`.
- The method must be deterministic.
- The method must not mutate any state.
- The implementation should be simple enough for OpenJML ESC to verify.
