# Min Of Three

## 问题描述

实现一个函数，输入三个整数 `a`、`b`、`c`，返回其中的最小值。

约定：

- 输入都是普通整数
- 返回值必须等于三个输入中的一个
- 返回值不大于任意输入

## 正确代码

```c
int min_of_three(int a, int b, int c) {
    int m = a;

    if (b < m) {
        m = b;
    }
    if (c < m) {
        m = c;
    }

    return m;
}
```

## 说明

这是最简单的多分支最值题，适合验证“逐步维护当前最小值”。
