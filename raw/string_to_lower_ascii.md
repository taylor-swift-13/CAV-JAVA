# String To Lower ASCII

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s`，把其中的大写英文字母原地转换为小写英文字母，其他字符保持不变。

约定：

- 输入是合法的 C 风格字符串
- 只处理 ASCII 大写字母 `'A'` 到 `'Z'`
- 函数原地修改字符串

## 正确代码

```c
void string_to_lower_ascii(char *s) {
    int i = 0;

    while (s[i] != '\0') {
        if (s[i] >= 'A' && s[i] <= 'Z') {
            s[i] = s[i] + 32;
        }
        i++;
    }
}
```

## 说明

这道题适合验证“字符串原地更新 + 字符范围变换”模式。
