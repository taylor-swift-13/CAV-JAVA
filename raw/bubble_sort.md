# Bubble Sort

## 问题描述

给定一个长度为 `n` 的整型数组 `a`，请使用冒泡排序将数组按非降序（从小到大）原地排序。

约定：
- 输入范围：`0 <= n <= 2000`。
- `a` 指向一段可写的连续数组内存。
- 数组至少包含 `n` 个元素。
- 元素范围：对所有 `0 <= i < n`，有 `INT_MIN <= a[i] <= INT_MAX`。
- 不申请额外堆内存，只在原数组上交换元素。

## 正确代码

```c
void bubble_sort(int *a, int n) {
    int i, j;

    for (i = 0; i < n; ++i) {
        for (j = 0; j + 1 < n - i; ++j) {
            if (a[j] > a[j + 1]) {
                int t = a[j];
                a[j] = a[j + 1];
                a[j + 1] = t;
            }
        }
    }
}
```

## 说明

这段代码使用双重循环实现冒泡排序：
- 外层循环控制已归位元素个数；
- 内层循环在未归位区间中比较相邻元素；
- 若 `a[j] > a[j + 1]`，就交换两者位置。

每轮内层循环结束后，当前未归位区间中的最大值会被“冒泡”到末尾。  
当外层循环结束时，整个数组按非降序排列。
