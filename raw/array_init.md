# Array Initialization

## 问题描述

给定一个非负整数 `n` 和一个长度至少为 `n` 的整型数组 `a`。

请把数组的前 `n` 个位置都初始化为 `0`。

约定：
- 输入 `n` 为非负整数。
- `a` 指向一段可写的连续数组内存。
- 数组至少包含 `n` 个元素。
- 只修改 `a[0]` 到 `a[n - 1]`，不访问越界位置。

## 正确代码

```c
void array_init(int n, int *a) {
    int i;

    for (i = 0; i < n; ++i) {
        a[i] = 0;
    }
}
```

## 说明

这段代码使用一个简单的顺序循环初始化数组：
- 循环变量 `i` 从 `0` 递增到 `n - 1`
- 每一步把当前位置 `a[i]` 赋值为 `0`
- 循环结束后，数组前 `n` 个元素都被初始化为 `0`

这是最基础的数组写入型算法题，后续写规格时通常需要表达：
- 输入数组在前置条件中是可写的
- 循环过程中前缀已经被置为 `0`
- 后置条件中前 `n` 个元素都等于 `0`
