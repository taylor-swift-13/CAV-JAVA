# String Starts With

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s` 和字符 `c`，判断字符串的第一个字符是否等于 `c`。

约定：

- 字符串是合法的 C 风格字符串
- 函数不修改字符串
- 如果 `s[0] == c`，返回 `1`
- 否则返回 `0`

## 正确代码

```c
int string_starts_with(char *s, char c) {
    if (s[0] == c) {
        return 1;
    }
    return 0;
}
```

## 说明

这道题适合验证“字符串首字符读取 + 简单分支”的模式。
