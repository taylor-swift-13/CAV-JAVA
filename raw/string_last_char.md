# String Last Char

## 问题描述

实现一个函数，输入一个非空字符串 `s`，返回它的最后一个字符。

约定：

- `s` 是合法的以 `'\0'` 结尾的字符串
- `s` 不是空串
- 函数不修改字符串

## 正确代码

```c
char string_last_char(char *s) {
    int i = 0;

    while (s[i + 1] != '\0') {
        i++;
    }

    return s[i];
}
```

## 说明

这道题是最基础的字符串尾字符扫描：

- 单层循环
- 不修改字符串
- 需要保证字符串非空
