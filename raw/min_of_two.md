# Min Of Two

Implement a Java method that returns the smaller of two integers.

Target method:

```java
public static int min_of_two(int a, int b)
```

Behavior:

- If `a <= b`, return `a`.
- If `a > b`, return `b`.

Requirements:

- The generated Java class should be named `MinOfTwo`.
- The method must be deterministic.
- The method must not mutate any state.
- The implementation should be simple enough for OpenJML ESC to verify.
