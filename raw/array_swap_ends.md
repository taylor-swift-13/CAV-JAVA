# Array Swap Ends

## 问题描述

实现一个函数，输入长度为 `n` 的整数数组 `a`。如果 `n >= 2`，交换首元素和尾元素；否则保持不变。

约定：

- `n >= 0`
- 数组长度恰好是 `n`
- 需要原地修改数组

## 正确代码

```c
void array_swap_ends(int n, int *a) {
    int t;

    if (n < 2) {
        return;
    }

    t = a[0];
    a[0] = a[n - 1];
    a[n - 1] = t;
}
```

## 说明

这道题适合验证“短分支 + 常量次更新”模式。
