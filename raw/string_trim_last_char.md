# String Trim Last Char

## 问题描述

实现一个函数，输入一个长度为 `n` 的字符串 `s`。如果 `n > 0`，删除最后一个字符；如果 `n == 0`，保持空串不变。

约定：

- 输入是合法的 C 风格字符串
- `n` 是字符串当前长度
- 需要原地修改字符串

## 正确代码

```c
void string_trim_last_char(int n, char *s) {
    if (n > 0) {
        s[n - 1] = '\0';
    }
}
```

## 说明

这道题适合验证“短分支 + 字符串末尾原地更新”模式。
