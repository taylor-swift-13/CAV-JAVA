# String Is Empty

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s`，判断它是否为空串。

约定：

- 如果 `s[0] == '\0'`，返回 `1`
- 否则返回 `0`
- 函数不修改字符串内容

## 正确代码

```c
int string_is_empty(char *s) {
    if (s[0] == '\0') {
        return 1;
    } else {
        return 0;
    }
}
```

## 说明

这是最简单的字符串判定题：

- 没有循环
- 一个分支
- 只读第一个字符
