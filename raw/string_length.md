# String Length

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s`，返回它的长度。

约定：

- `s` 指向合法的以 `'\0'` 结尾的字符数组
- 函数不修改字符串内容
- 返回值不包括结尾的 `'\0'`

## 正确代码

```c
int string_length(char *s) {
    int i = 0;

    while (s[i] != '\0') {
        i++;
    }

    return i;
}
```

## 说明

这是最基础的字符串扫描题：

- 单层循环
- 只读字符串
- 退出条件由 `'\0'` 决定
