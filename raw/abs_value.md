# Absolute Value

## 问题描述

实现一个函数，输入一个整数 `x`，返回它的绝对值。

约定：

- 输入满足 `x != INT_MIN`
- 这样 `-x` 仍然在 32 位有符号整数范围内
- 函数不读写任何额外内存

## 正确代码

```c
int abs_value(int x) {
    if (x >= 0) {
        return x;
    } else {
        return -x;
    }
}
```

## 说明

这道题比简单加法多一个符号翻转分支：

- 没有循环
- 没有数组
- 需要在 contract 中显式排除 `INT_MIN`
- 后置条件适合写成“返回值非负，且等于 `x` 或 `-x`”
