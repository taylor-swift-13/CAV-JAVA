# String To Upper ASCII

## 问题描述

实现一个函数，输入一个以 `'\0'` 结尾的字符串 `s`，把其中所有小写字母转成大写字母，其他字符保持不变。

约定：

- 输入是合法的 C 风格字符串
- 需要原地修改字符串

## 正确代码

```c
void string_to_upper_ascii(char *s) {
    int i = 0;

    while (s[i] != '\0') {
        if ('a' <= s[i] && s[i] <= 'z') {
            s[i] = s[i] - 'a' + 'A';
        }
        i++;
    }
}
```

## 说明

这道题适合验证“字符串扫描 + 条件原地更新”模式。
